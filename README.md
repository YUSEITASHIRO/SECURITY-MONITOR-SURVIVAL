## 📖 世界観 (Story)

深夜の静寂に包まれた極秘施設。
あなたは一晩の「監視代行」を請け負った臨時警備員です。

支給されたのは、古びた監視モニターと、施設内の各所に設置された10台のカメラへのアクセス権。
しかし、この業務には一つだけ、絶対に守らなければならない「特記事項」がありました。

**『影を視界から外してはならない』**

施設内を徘徊する「何か」は、誰かに観測されている間だけ、その実体をこの世界に繋ぎ止めることができます。
もし、モニターを切り替え損ねたり、カメラが故障したりして、奴への視線が途切れてしまったら——。
奴は観測者を「唯一の出口」と認識し、あなたのいる監視室へと一瞬で転移してくるでしょう。

朝が来るまで、あと5つのシフト。あなたは、最後まで「観測」を続けられますか？

## 🎯 設計思想 (Design Philosophy)

### 1. 「観測」による抑止の緊張感
本作の防衛手段は攻撃でも回避でもなく、**ただ見続けること**です。プレイヤーに「安全を確認するために画面を見る」のではなく、「死なないために画面を見続けなければならない」という逆転した心理状態を強いることを意図しています。

### 2. 視界の外の不確定性
監視カメラには当然、死角が存在します。モンスターが角を曲がった瞬間にどのカメラに切り替えるべきか、あるいは今のノイズは故障かそれとも奴の接近か。限られた情報の断片から状況を推測する際の不快な緊張感を重視して設計しました。

### 3. アナログ感のあるUI演出
あえて解像度を落としたモニター映像、走査線、そして突如として発生する「カメラ故障」。デジタルの正確性と、アナログの不安定さを融合させることで、プレイヤーが操作するデバイスそのものへの不信感を煽り、没入感を高めています。

## 🔧 ゲームシステム (Systems Design)

### コアループ
1. **Ready**: 生成されたエリアのカメラ配置を確認し、戦略を立てる。
2. **Monitoring**: 1〜9のチャンネルを切り替え、モンスターを視界（FOV）に収め続ける。
3. **Grace Timer**: 視界から外れた瞬間、猶予時間のカウントを開始。ゼロになれば転移（Scare）が発生。
4. **Shift Clear**: 30秒間の監視任務を完遂すれば次のシフトへ。全5シフトで終了。

### ダイナミック・カメラ・インテリジェンス
カメラシステムは単なる画像表示ではありません。
- **FOV判定**: モンスターの座標をカメラのビュー行列で射影し、実際に画面内に映っているかを毎フレーム精密に計算します。
- **故障イベント**: ターン2以降、低確率でカメラが深刻な「信号消失（砂嵐）」を引き起こします。故障中はそのチャンネルは無力化され、他のカメラでの索敵を余儀なくされます。

## 🕹 操作方法 (Controls)

| 操作 | アクション |
|------|-----------|
| `0 〜 9 キー` | 監視カメラ（CH0〜CH9）の切り替え |
| `矢印キー (Arrow Keys)` | 監視カメラの向きを左右に回転・調整 |
| `Y キー` | 準備完了、監視シフトを開始する |
| `Enter` | タイトル画面でのスタート / メッセージの送信 |

## 🛠 技術スタック (Technical Stack)

- **言語**: C# / .NET 9.0
- **グラフィック**: Raylib-cs (OpenGL 3.3 ラッパー)
- **アルゴリズム**:
  - **Procedural Hallway Generation**: BFS（幅優先探索）を用いた接続グラフにより、毎ターン異なる監視エリアを生成。
  - **Frustum Culling Logic**: 3D座標のビューポート射影による、モンスターの視認判定。
  - **RenderTexture Multiplexing**: 監視カメラの映像を動的にテクスチャ化し、監視室のモニターにマッピングする手法。

## 📝 制作メモ (Retrospective)

当初は「逃げ回るホラー」を想定していましたが、開発初期に実装した「監視カメラを切り替える」という動作の面白さに注目し、ジャンルを「監視シミュレーター」へとシフトしました。
特に苦労したのは「カメラが正しくモンスターを捉えているか」の判定ロジックです。単なる距離判定ではなく、障害物や壁越しでは見えないという当たり前の物理法則を、軽量な数学的計算（ドット積と線分判定）で実現しました。
この「限定的な情報の中で最善を尽くす」という体験が、本作の価値であると思っています。
